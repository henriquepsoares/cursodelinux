# 🐧 Introdução ao Linux

![Tux - Mascote do Linux](https://upload.wikimedia.org/wikipedia/commons/a/af/Tux.png)

O **Linux** é um sistema operacional **livre e de código aberto**, baseado no kernel Linux, que serve como núcleo para inúmeras distribuições (distros) como **Ubuntu, Fedora, Debian e CentOS**. Ele é conhecido por sua **estabilidade, segurança e flexibilidade**, sendo usado desde desktops e servidores até supercomputadores e dispositivos embarcados.

---

## 🚀 Por que usar Linux?

- **Gratuito e Open Source** – Você pode usar, modificar e redistribuir sem custos.  
- **Segurança e estabilidade** – Menos vulnerável a vírus e falhas.  
- **Flexibilidade e personalização** – Configure do kernel à interface gráfica.  
- **Amplo suporte da comunidade** – Tutoriais, fóruns e contribuições de milhares de desenvolvedores.  
- **Usado profissionalmente** – Presente em servidores, nuvem, IoT e supercomputadores.

---

## 🐧 Componentes principais do Linux

| Componente         | Descrição                                      |
|--------------------|------------------------------------------------|
| **Kernel**         | Núcleo do sistema que gerencia hardware e processos. |
| **Shell**          | Interface de linha de comando para interação com o sistema. |
| **Sistema de arquivos** | Organização dos dados no disco.                  |
| **Utilitários**    | Ferramentas para tarefas comuns, como manipulação de arquivos. |

---

## 💡 Comandos básicos do Linux

| Comando  | Função                                | Exemplo                          |
|----------|-------------------------------------|---------------------------------|
| `ls`     | Lista arquivos e diretórios         | `ls -la`                        |
| `cd`     | Muda o diretório atual               | `cd /home/usuario`              |
| `cp`     | Copia arquivos                      | `cp arquivo.txt /tmp`           |
| `mv`     | Move ou renomeia arquivos           | `mv arquivo.txt novo.txt`       |
| `rm`     | Remove arquivos                     | `rm arquivo.txt`                |
| `chmod`  | Altera permissões de arquivos       | `chmod +x script.sh`            |
| `grep`   | Pesquisa texto dentro de arquivos   | `grep "erro" arquivo.log`       |
